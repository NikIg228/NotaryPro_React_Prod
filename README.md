# Notary

Минималистичный интерфейс для создания нотариальных документов в стиле ChatGPT.

## Технологии

- **React 18** + **TypeScript**
- **Vite** - сборщик проекта
- **React Router** - навигация
- **TailwindCSS** - стилизация
- **React Hook Form** + **Zod** - формы и валидация (готово к использованию)

## Структура проекта

```
src/
├── components/       # UI компоненты
│   ├── Button.tsx
│   ├── Card.tsx
│   ├── DocumentCard.tsx
│   ├── Filters.tsx
│   ├── Header.tsx
│   ├── Input.tsx
│   └── Search.tsx
├── data/            # Данные документов
│   └── documents.json
├── pages/           # Страницы приложения
│   ├── HomePage.tsx
│   └── CategoryPage.tsx
├── utils/           # Утилиты
│   └── dataUtils.ts
├── App.tsx          # Главный компонент
├── main.tsx         # Точка входа
└── index.css        # Глобальные стили
```

## Запуск проекта

```bash
# Установка зависимостей
npm install

# Запуск dev сервера
npm run dev

# Сборка для production
npm run build

# Просмотр production сборки
npm run preview
```

## Особенности дизайна

Интерфейс выполнен в минималистичном стиле ChatGPT:

- **Белый фон** с серыми акцентами
- **Фиолетовый акцентный цвет** для интерактивных элементов
- **Большие отступы** и много воздуха
- **Упрощённые компоненты** без тяжёлой графики
- **Мягкие hover эффекты** на карточках
- **Чёткая типографика**: полужирные заголовки, серый текст

## Страницы

1. **Главная страница** (`/`) - список категорий документов
2. **Страница категории** (`/category/:categoryId`) - список документов в категории с поиском и фильтрами

## Данные

Документы хранятся в `src/data/documents.json`. Структура данных сохранена без изменений из оригинального `docs_structure.txt`.

## Стили

Все компоненты используют TailwindCSS с кастомными цветами в стиле ChatGPT:
- `bg-white` - основной фон
- `bg-gray-50` - серые разделы
- `border-gray-200` - границы
- `text-gray-900` - заголовки
- `text-gray-600` - основной текст
- `black` - акцентный цвет кнопок и активных элементов
